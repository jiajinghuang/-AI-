## 📖 应用部署手册 v1.0

---

## 一、系统概述

### 1.1 项目简介

**智评星**是一套面向小学综合学科评价的轻量级Web应用系统，专为解决"双减"背景下七大学科（英语、音乐、美术、体育、科学、心理、信息科技）无纸笔考试评价难题而设计。

系统采用"前端静态页面 + 云端数据中台"架构，无需服务器部署，只需三个HTML文件即可运行，配合QuickForm数据平台实现数据的云端存储与统计分析。

### 1.2 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                        智评星系统                            │
├──────────────┬──────────────┬───────────────────────────────┤
│  科任老师    │   班级答题   │      数据大屏监控端            │
│   出题端     │      端      │                                │
├──────────────┴──────────────┴───────────────────────────────┤
│                    QuickForm 云端数据平台                    │
│         (题库存储 / 答卷存储 / 数据聚合 / 可视化分析)         │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 模块功能

| 模块名称 | 使用对象 | 核心功能 |
|---------|---------|---------|
| 科任老师出题端 | 七大学科教师 | 录入题目、上传图片、提交至云端题库 |
| 班级趣味答题端 | 学生（班级为单位） | 抽取题目、互动答题、提交答卷数据 |
| 多维数据大屏监控端 | 教务/管理者 | 查看出题进度、答题统计、学情分析 |

## 三、QuickForm平台配置

> 💡 **什么是QuickForm？**
> QuickForm是一个免费的云端数据收集平台，可为你的网页提供数据存储接口（API），无需自己搭建服务器。

### 3.1 注册账号

1. 访问官网：**https://quickform.cn/**
2. 点击右上角【注册/登录】
3. 使用手机号或邮箱完成注册（过程略）

> 📌 **提示**：建议使用学校邮箱注册，教师认证后可创建更多任务。

### 3.2 创建数据任务

本系统需要创建**两个**数据任务：

- **任务一**：题库数据（存储老师提交的题目）
- **任务二**：答卷数据（存储学生答题记录）

#### 步骤一：创建题库任务

1. 登录后进入控制台，点击【新建任务】

![新建任务按钮](截图/01_新建任务按钮.png)

2. 填写任务名称，如：`智评星_题库`

3. 数据字段会自动按JSON格式存储，无需手动逐个创建字段，字段格式如下：

```json
{
  "年级": "一年级",
  "学科": "音乐",
  "周数": "第8周",
  "题目数量": 5,
  "题目数据": [
    {
      "题序": 1,
      "题干": "快乐的农夫 作曲家是以下",
      "正确答案": "A",
      "题型": "单选题",
      "选项": {
        "A": "舒曼",
        "B": "贝多芬"
      }
    }
    // ...更多题目
  ],
  "提交时间": "2026-04-23 15:26:31",
  "覆盖": false
}
```

> 📌 **注意**：出题端会以这种JSON格式一次性提交一组题目（5题），而不是逐题提交。

4. 点击【创建任务】完成创建

#### 步骤二：创建答卷任务

重复上述步骤，创建第二个任务，命名为：`智评星_答卷`

答题数据的JSON格式如下：

```json
{
  "classInfo": "二年级1班",
  "grade": "二年级",
  "classNum": "1班",
  "week": "第6周",
  "statistics": {
    "音乐": {"total": 5, "correct": 4},
    "体育": {"total": 5, "correct": 5},
    "美术": {"total": 5, "correct": 5},
    "心理": {"total": 5, "correct": 5},
    "科学": {"total": 5, "correct": 5}
  },
  "subjectAnswers": {
    "音乐": [
      {"题序": 1, "学生答案": "B", "正确答案": "B", "是否正确": true, "题型": "单选题"}
      // ...更多题目
    ],
    "体育": [...],
    "美术": [...],
    "心理": [...],
    "科学": [...]
  },
  "totalCorrect": 24,
  "totalQuestions": 25,
  "submitTime": "2026-04-10T05:35:24.659Z"
}
```

> 📌 **注意**：答题端会以这种JSON格式一次性提交整个班级的答题结果。

### 3.3 获取API地址

每个任务创建成功后，系统会生成专属的API地址。

1. 在任务列表中，点击目标任务进入详情页

2. 找到【API接口地址】

![获取API地址](截图/05_获取API地址.png)

3. API地址格式如下：

| 接口类型 | 地址格式 | 用途 |
|---------|---------|------|
| POST提交接口 | `https://quickform.cn/api/xxxxxxxx` | 用于提交数据 |
| GET查询接口 | `https://quickform.cn/api/xxxxxxxx/all` | 用于查询/获取数据 |

> ⚠️ **重要**：
> - **出题端** 需要用到 **POST提交接口**
> - **答题端** 需要用到 **POST提交接口**
> - **数据大屏** 需要用到 **GET查询接口（/all）** 来读取已提交的数据
>
> 请分别复制"题库任务"和"答卷任务"的API地址，下一步配置时需要用到。

## 四、系统部署配置

### 4.1 配置出题端

1. 用文本编辑器（推荐 VS Code 或 记事本）打开 `出题端.html`

2. 找到API配置位置（代码中会有注释标注）

![出题端API配置](截图/02_出题端API配置.png)

3. 将找到的地址替换为你在QuickForm创建的**题库任务API地址**（POST接口）

   ```javascript
   // 替换前（示例）
   // const QUESTION_API = "";  // TODO: 在这里填写题库API地址
   
   // 替换后
   const QUESTION_API = "https://quickform.cn/api/你的题库任务ID";
   ```

4. 保存文件

### 4.2 配置答题端

1. 打开 `答题端.html`

2. 找到API配置位置

![答题端API配置](截图/03_答题端API配置.png)

3. 分别替换为对应地址：

   ```javascript
   // 题库API（用于获取题目 - GET查询接口）
   const QUESTION_API = "https://quickform.cn/api/你的题库任务ID/all";
   
   // 答卷API（用于提交答题数据 - POST提交接口）
   const ANSWER_API = "https://quickform.cn/api/你的答卷任务ID";
   ```

4. 保存文件

### 4.3 配置数据大屏

1. 打开 `数据大屏.html`

2. 找到API配置位置

![数据大屏API配置](截图/04_大屏API配置.png)

3. 替换题库API和答卷API地址（大屏需要同时读取两个数据源）：

   ```javascript
   // 题库API（GET查询接口，用于获取题目列表）
   const QUESTION_API = "https://quickform.cn/api/你的题库任务ID/all";
   
   // 答卷API（GET查询接口，用于获取答题统计数据）
   const ANSWER_API = "https://quickform.cn/api/你的答卷任务ID/all";
   ```

4. 保存文件

## 五、使用指南

### 5.1 出题端使用

**适用场景**：各科任老师在双周趣味问答前录入题目

**操作流程**：

1. 双击打开 `出题端.html`（或拖入浏览器）

2. 选择年级、学科和周次

3. 填写一组题目（5道）：

   - 输入每道题的题干
   - 输入选项内容
   - 选择正确答案
   - （可选）如需添加图片，参考下方"图片上传说明"

4. 点击【提交题目】

5. 提示成功后，题目将存入云端题库

**图片上传说明**：

如需添加配图（如音乐音频链接、美术名画链接等）：

1. 使用图床工具（如 [图床小镇](https://imgbed.cn/)）上传图片
2. 获取图片的公开访问链接
3. 将链接粘贴至题目中即可

> 📌 **推荐图床**：
> - 图床小镇（imgbed.cn）👍
> - sm.ms
> - 路过图床（imgtu.com）

### 5.2 答题端使用

**适用场景**：综合课或班会课上，全班集体答题

**操作流程**：

1. 在班级一体机上打开 `答题端.html`

2. 选择年级和班级

3. 系统自动加载当前年级、当前周次的题目

4. 学生讨论作答，点击选项提交

5. 答题完成后，系统自动将数据提交至云端

6. 界面显示答题结果和正确答案

### 5.3 数据大屏使用

**适用场景**：教务管理、教研分析、全校数据展示

**操作流程**：

1. 打开 `数据大屏.html`

2. 系统自动加载云端数据，展示以下信息：

   - **出题进度**：各学科是否已完成本周出题
   - **答题统计**：各班级参与情况、正确率
   - **学情分析**：错误率较高的知识点定位

3. 可按周次、学科、班级筛选查看

## 六、常见问题 FAQ

### Q1：提交题目时提示"网络错误"？

**可能原因**：
- API地址未正确配置
- 网络连接问题
- QuickForm平台临时不可用

**解决方案**：
1. 检查API地址是否正确复制（注意不要有多余空格）
2. 检查浏览器控制台（F12）查看具体错误信息
3. 尝试直接访问API地址测试连通性

### Q2：答题端加载不出题目？

**可能原因**：
- 题库为空（该年级/周次尚未出题）
- 题库API地址配置错误
- API地址缺少 `/all` 后缀

**解决方案**：
1. 先在出题端录入题目
2. 确认年级和周次匹配
3. 检查API地址是否为GET查询接口（带 `/all`）

### Q3：数据大屏显示空白？

**可能原因**：
- QuickForm数据接口有访问限制
- API地址配置为POST接口而非GET接口

**解决方案**：
1. 确认大屏使用的是 `/all` 后缀的GET接口
2. 确认QuickForm任务设置为"公开"状态
3. 如仍有问题，联系QuickForm官方支持

### Q4：如何导出数据？

1. 登录 QuickForm 控制台
2. 进入对应任务
3. 点击【导出数据】
4. 选择 Excel 或 CSV 格式下载

### Q5：可以同时运行多个学校/年级吗？

可以！建议：
- 为不同学校创建独立的QuickForm账号
- 或在同一账号下创建多套任务（如：智评星_题库_学校A）
